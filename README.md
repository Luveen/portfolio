# Mishan Luveen - Portfolio 🚀

Welcome to my personal developer portfolio! This project is a showcase of my work, designed with a premium, Apple-inspired frosted glass aesthetic. It features a sleek, high-contrast monochromatic dark mode with vibrant orange accents to create a modern studio feel. Built natively with Vue 3, the site incorporates dynamic 3D-feeling layouts like a "Convex Ribbon" archive for my projects, and a unique, interactive terminal-style floating CLI acting as the main navigation hub. The entire experience is fully responsive and connected to a live cloud database to serve fluid, real-time data seamlessly out of the box.
